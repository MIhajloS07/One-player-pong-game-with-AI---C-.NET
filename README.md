# PONG Game with AI (C# Windows Forms Application)

## Controls

- **Move Paddle:** `RIGHT` / `LEFT` arrows  
- **Pause/Resume Game:** `ESCAPE` key  
- **Interaction Menu:** Enter the number of points to play against the AI/Bot  

> **Note:** If you press **Cancel** or enter an invalid value in the interaction box, the game will automatically set the points to the default value of **5**.  

## AI / Bot Model

- The AI is designed to be **beatable**, providing a fair challenge for the player.
